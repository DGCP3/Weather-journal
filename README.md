# Weather Journal

## Run Locally

Clone the project

```bash
  git clone https://github.com/DGCP3/weather-journal.git
```

Go to the project directory

```bash
  cd /weather-journal
```

Install dependencies

```bash
  npm install
```

Create .env with PORT variable start the server

```bash
  npm run start
```
